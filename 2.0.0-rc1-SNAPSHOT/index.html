Keypop Card C++ API Documentation

Overview

The Keypop Card C++ API defines interfaces for card communication aligned with the Terminal Card specifications from the Calypso Networks Association. This C++ interface provides a port of the reference Keypop Card Java API.

Architecture & Components

APDU Communication

  • keypop::card::spi::ApduRequestSpi Interface for constructing and formatting APDU commands
  • keypop::card::ApduResponseApi Interface for handling and processing APDU responses
  • keypop::card::spi::CardRequestSpi Manager for grouping multiple APDU commands
  • keypop::card::CardResponseApi Container for responses from multiple APDU executions

Card Selection Process

  • keypop::card::spi::CardSelectionRequestSpi Input data and parameters for card selection operations
  • keypop::card::CardSelectionResponseApi Card initialization and selection data handler
  • keypop::card::spi::CardSelectionExtensionSpi Extension mechanisms for specific card protocols
  • keypop::card::spi::SmartCardSpi Foundation interface for card implementations

Communication Channel Management

  • keypop::card::ProxyReaderApi Request transmission and channel control handler
  • keypop::card::ChannelControl Channel management policies (KEEP_OPEN, CLOSE_AFTER)

Exception Handling

The API implements the following exception hierarchy:

  • keypop::card::AbstractApduException Base class for APDU-related exceptions
  • keypop::card::CardBrokenCommunicationException Card communication failure handling
  • keypop::card::ReaderBrokenCommunicationException Reader communication failure handling
  • keypop::card::UnexpectedStatusWordException Unexpected APDU status response handling
  • keypop::card::ParseException Card selection response parsing failure handling
